Wearable Sensors

Pretraining

PAMAP-2

The PAMAP-2 dataset is a physical activity monitoring dataset with data for 18 different physical activities. Examples of activities include walking, cycling, playing soccer, etc. Data was collected from 9 subjects performing the activities while wearing 3 inertial measurement units and a heart rate monitor. Uses of the dataset include activity recognition and intensity estimation. Algorithms of data processing, segmentation, feature extraction and classification can be developed and applied using the dataset.

Transfer

PAMAP-2

The transfer task is to predict the activity labels (e.g. "walking," "cycling," or "playing soccer") from the sensor recordings.
