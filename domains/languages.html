mC4 Dataset

We use a subset of the large mC4 dataset (English, French, German, Spanish, Chinese, Japanese, and Korean). This is a cleaned version of Common Crawl's web crawl corpus.

Transfer

We use the PAWS-X datasets to finetune and assess the pretrained multilingual models. These include paraphrase identification tasks in 7 distinct languages (French, Spanish, German, Chinese, Japanese, and Korean).

PAWS-X

The original Paraphrase Adversaries from Word Scrambling (PAWS) dataset consists of English sentence pairs curated from Wikipedia and Quora. The PAWS-X dataset contains translations of the PAWS dataset into the languages listed above. This makes for a total of roughly 50,000 examples across each of the 7 languages.
